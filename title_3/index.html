Project Description

The algorithm generates different geometric outputs based on the input values.
Input Value G (greenery): a green mesh is created based on the existing plot boundaries.
Input Value P (public space): a grey mesh is generated based on the existing plot boundaries.
Input Value I (industrial): the plot boundaries are offseted by 2 meters and an extruded and caped mesh is drawn based on the offseted lines.
Input Value C (city - mixed use): the outlines of each plot are offset by the value 2 meters. If the length of each offsetted line is bigger than a set value the lines are offsetted once again. The next step generates the extruded geometry by subtracting the area enclosed by the inner lines, in order to recreate the look of a block perimeter development.
Input Value R (residential): this function divides the assigned plot into new smaller plots. Those plots are offsetted once again and the new geometry is extruded.

The current challanges include finding the smaller plots for the value R, as well as chosing the right length to offset depending on the plot size for the Value C.
The visualization is especially valuable for the enduser, by refering to the size of the cultivated area, more efficient geometries can be generated based on each individual plot. Therefore the area can be used in the most efficient way.
